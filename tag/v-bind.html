v-bind 示例（Vue 3）

基本属性绑定（: 或 v-bind:）
访问 Vue 官网（:href） 
<a :href="url">... <img :src="logo" :alt="imageAlt">
布尔属性绑定（disabled/required 等） 
<button :disabled="isDisabled">
class 绑定（对象 / 数组 / 字符串）
对象语法
数组语法
计算属性（字符串）
<div :class="{ active: isActive }"> 或 <div :class="[classA, classB]">
style 绑定（对象 / 数组）
对象 style
数组 style
<div :style="{ color: activeColor }">
动态参数（动态属性名） 
v-bind:[attrName]="attrValue"
将对象全部绑定到元素（v-bind="object"） 
v-bind="attrs" （将对象中的键值全部作为属性绑定）
绑定组件 Props（:prop / v-bind） 
<child :label="childProps.label" :count="childProps.count" /> 或 <child v-bind="childProps" />
.prop 修饰符（直接设置 DOM property） 
<input type="checkbox" :checked.prop="isChecked">
数据属性与 ARIA（data-*, aria-*） 
<button :aria-label="ariaLabel" :data-index="index">
