v-if 与 v-show：区别与对比

下面两个卡片分别演示用 `v-if` 和 `v-show` 控制元素显示：一个通过销毁/创建 DOM（`v-if`），一个通过 CSS `display` 隐藏（`v-show`）。

v-if 示例（DOM 会被创建/销毁）

这是 v-if 渲染的内容 —— 每次变为 false 时会被移除。

输入内容：{{ textIf }}

v-if 当前为 false，内容被移除。

v-show 示例（通过 CSS display 隐藏）

这是 v-show 渲染并始终保留在 DOM 中的内容。

输入内容：{{ textShow }}

虽然隐藏，但 DOM 仍存在（检查页面元素可见）。

要点

  • 当频繁切换且需要保留状态时，优先使用 v-show（开销小，切换仅变更样式）。
  • 当条件很少变更或条件为 false 时不应占用 DOM 时，使用 v-if（会销毁/重建）。
